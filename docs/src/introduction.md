# Introduction

This documentation aims to provide up-to-date getting started guide on how to use the `intasend` crate with your
client and backend applications.

This crate covers:

1. Collections API
2. Checkouts API
3. Wallets API
4. Payouts API
5. Refunds API
6. Payment Links API
7. ~Collection customers~
8. ~Subscription & subscription plans~
